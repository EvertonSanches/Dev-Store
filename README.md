# Dev-Store
Projeto desenvolvido com intuito de aprender sobre animações com CSS e JS, controle de estados de interface e responsividade com media queries.
Também feito com intuito de "preparar" para os estudos de React.

Tudo que aprendi neste projeto:

Controle de estado em JavaScript puro
Uso de classes CSS para representar estados da interface
Animações acionadas por mudança de estado
Transições com "transform" e "opacity"
Organização de código pensando em UI reativa
Responsividade com media queries
Ajustes básicos de performance no mobile
